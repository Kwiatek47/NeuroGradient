# Pomodoro Timer

Prosta aplikacja webowa do techniki Pomodoro z React frontend i Node.js backend.

## Uruchomienie z Docker Compose

```bash
docker-compose up --build
```

Aplikacja będzie dostępna na http://localhost:3000

## Instalacja lokalna

### Backend
```bash
cd backend
npm install
npm start
```

### Frontend
```bash
cd frontend
npm install
npm start
```

## Funkcje

- Timer Pomodoro (25 min pracy, 5 min krótka przerwa, 15 min długa przerwa)
- Wysuwane menu z wyborem trybu
- Prosty i czytelny interfejs
